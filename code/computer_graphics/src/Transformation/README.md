# Transformation
> Transformatino is process of modifying and re-arranging the grapics.


## Types of transformation:-
2D Transformation
3D Transformation


### 2D Transformation
> In 2d Transformation we modify 2D graphics.

#### Types of 2D Transformation
1. Translation
2. Rotation
3. Scaling
4. Reflection
5. Shearing
