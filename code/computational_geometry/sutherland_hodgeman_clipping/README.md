## Working of Sutherland Hodgeman Polygon Clipping

**Coordinates of rectangular clip window** :

xmin,ymin             :`200 200`

xmax,ymax             :`400 400`


**Polygon to be clipped** :

Number of sides       :`3`

Enter the coordinates :

`150 300`

`300 150`   

`300 300`
